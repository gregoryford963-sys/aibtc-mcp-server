# stx402-agent

A Claude plugin / agent that provides scripts and helpers Claude can use to discover, pay for, and execute x402 endpoints from a user’s own Claude instance using a user-provided wallet.

Users install the plugin, provide their wallet, and the agent uses x402 endpoints to access yield optimization and other on-chain services, effectively allowing the agent to manage idle funds on the user’s behalf.

This project focuses on consumer-side tooling only.
x402 endpoints are created and maintained separately.
